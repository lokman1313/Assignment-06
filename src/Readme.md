# DigiTools

DigiTools is a React-based web application that showcases and manages a collection of premium digital products.
Users can browse available tools, add them to a cart, view selected items, and complete a simulated checkout process.

## 🚀 Features
- Add/remove products to/from cart.
- Dynamic cart counter in navbar.
- Conditional UI rendering. etc.

## 🛠️ Technologies Used
- React.
- Tailwind CSS.
- JavaScript (ES6).
- DaisyUI.
- Vite.
